# A Fan Page

An interactive fan-page prototype built with Vite, TypeScript, Three.js, and
WebGL. The experience includes a gesture-driven reveal, a layered portrait
parallax scene, interactive archive hotspots, and a ping-pong framebuffer
fluid cursor.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The production build can be published from `dist` to a GitHub Pages branch.
